# Icons Directory

This directory should contain the following icon files for the Grammar Bot Chrome extension:

- `icon16.png` - 16x16 pixels (toolbar icon)
- `icon48.png` - 48x48 pixels (extension management page)
- `icon128.png` - 128x128 pixels (Chrome Web Store)

## Icon Requirements

- **Format**: PNG
- **Style**: Modern, clean design representing grammar/writing
- **Colors**: Should work well on both light and dark backgrounds
- **Transparency**: Use transparent backgrounds

## Creating Icons

You can:
1. Design your own icons using tools like Figma, Sketch, or Canva
2. Use free icon sets from sites like:
   - [Feather Icons](https://feathericons.com/)
   - [Heroicons](https://heroicons.com/)
   - [Phosphor Icons](https://phosphoricons.com/)
3. Generate icons using AI tools like DALL-E, Midjourney, or Stable Diffusion

## Suggested Design

- Main element: Checkmark, pen, or document
- Secondary elements: Could include sparkles/stars to indicate AI
- Color scheme: Blue (#2d5aa0) to match the extension theme

## Temporary Solution

Until you add proper icons, you can:
1. Create simple colored squares as placeholders
2. Use any existing PNG files renamed to the required names
3. The extension will still work without icons, but Chrome may show warnings
